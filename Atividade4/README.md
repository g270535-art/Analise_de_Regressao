## Instruções gerais

Baixe o arquivo `Atividade04.zip`.

Utilize o arquivo `Atividade04.Rmd` para inserir as respostas. 

Utilize `echo=FALSE`, nenhum comando/mensagem/saída sem formatação deve estar aparente 
no arquivo que será gerado a partir do `.Rmd`. Quando o código precisa estar explícito, o exercício irá mencionar.

Todas as respostas numéricas devem ser inseridas no texto com comandos do R (reprodutíveis).

Todos os gráficos devem conter legendas (*captions*) claras e informativas, 
indicando o que está sendo analisado e destacando os principais padrões ou conclusões observadas. 

Todas as tabelas devem ser enumeradas e conter legenda. 

Confiram exemplos simples de formatação para gráficos e tabelas no [template](https://drive.google.com/drive/folders/14EHl83avFMdYPQMJSe76tGIetdn5Y6oJ?usp=sharing).


**USO DE IA**

Uso Ético: A IA pode ser usada para potencializar o aprendizado (inspirar, estruturar, formatar), e não para substituir a produção de pensamento crítico pelo discente.



## Envio das respostas


Submeta um arquivo `.zip` com os arquivos necessários para que `Atividade04.Rmd` rode sem erros e gere o `.pdf` com suas respostas.


**Inclua um arquivo** de texto listando as versões das plataformas de IA utilizadas e cópia exata dos prompts utilizados em cada uma, se for o caso (ou o link do compartilhamento do prompt).
